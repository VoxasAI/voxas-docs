## When to use this

Use this guide when you want more predictable and higher-quality assistant behavior in Voxas.

<Note>Image placeholder: Prompt or skill editor panel in Voxas.</Note>

## User flow

1. Define the task clearly.
2. Add constraints or style requirements.
3. Run and refine based on output.

## How to do it

### Step 1: Start with a concrete goal

Write the exact outcome you want in one sentence.

### Step 2: Add context and constraints

Include relevant files, assumptions, and limits (for example, timeframe or output format).

### Step 3: Iterate

If output is off-target, refine the prompt with one specific change at a time.

## What success looks like

- Responses match your goal and constraints.
- You need fewer retries to get useful output.

## What could go wrong

### Issue: Responses are too generic

- **Why it happens:** Prompt lacks specific context.
- **How to fix:** Add concrete inputs and expected output format.

### Issue: Assistant ignores part of the request

- **Why it happens:** Multiple tasks are bundled in one long prompt.
- **How to fix:** Split into smaller sequential prompts.
