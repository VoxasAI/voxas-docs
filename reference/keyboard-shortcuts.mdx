## When to use this

Use these shortcuts during active sessions to keep momentum without breaking focus.

<Note>Image placeholder: Shortcut cheat sheet overlay in Voxas.</Note>

## Available shortcuts

| Shortcut    | Action                   |
| ----------- | ------------------------ |
| `Cmd+\`     | Toggle window visibility |
| `Cmd+T`     | Toggle dark/light theme  |
| `Cmd+Up`    | Move session room up     |
| `Cmd+Down`  | Move session room down   |
| `Cmd+Left`  | Move session room left   |
| `Cmd+Right` | Move session room right  |

## What success looks like

- Shortcuts trigger immediately while Voxas is active.
- You can reposition or toggle views without mouse navigation.

## What could go wrong

### Issue: Shortcut does not trigger

- **Why it happens:** OS conflict or app focus issue.
- **How to fix:** Ensure Voxas is active and test another combination.

### Issue: Shortcut conflicts with another app

- **Why it happens:** Global keybinding overlap.
- **How to fix:** Temporarily disable conflicting app shortcut.
