## When to use this

Use this guide if you are installing Voxas for the first time on macOS.

<Note>
  Image placeholder: Voxas download page with the desktop build highlighted.
</Note>

## User flow

1. Download the latest Voxas Desktop build.
2. Move Voxas to `Applications`.
3. Launch the app and pass macOS security prompts.

## How to do it

### Step 1: Download the app

Download the latest desktop release from the Voxas distribution link shared by your team.

### Step 2: Install to Applications

Drag the Voxas app to the `Applications` folder.

### Step 3: Open Voxas

Open Voxas from `Applications`. If macOS warns about opening the app, approve the dialog and continue.

## What success looks like

- Voxas opens and shows the sign-in flow.
- No install-related errors block startup.

## What could go wrong

### Issue: App cannot open

- **Why it happens:** macOS security restrictions block first launch.
- **How to fix:** Open System Settings and allow the app, then launch again.

### Issue: App launches but closes immediately

- **Why it happens:** Incomplete install or corrupted app package.
- **How to fix:** Delete Voxas from `Applications`, reinstall from a fresh download, then retry.
