description: "A desktop AI copilot for technical interview workflows."
---

Voxas is a desktop app that helps you run technical interview workflows with real-time AI assistance. You can manage sessions, chat with the assistant, and use shortcuts to stay fast during active problem-solving.

<Note>
  Image placeholder: Hero screenshot of Voxas desktop app home screen.
</Note>

## Who this is for

- Engineers preparing for coding interviews.
- Users who want a desktop-first AI assistant while working through coding tasks.
- Teams piloting structured interview prep workflows.

## Core user flow

1. Install Voxas Desktop.
2. Sign in and grant required macOS permissions.
3. Start your first session.
4. Use chat, session history, and shortcuts during your workflow.
